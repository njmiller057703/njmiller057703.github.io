Part 1: Perspective and Orthographic Projections

The first part of the project was to emphasize the differences between perspective and orthographic projections. The first figure displays the orthographic projection because of the parallel lines seen in the pillars and the levels of each floor. Figure two shows the perspective projection emphasizing the lines collapsing towards a horizon. The lines on the brick path as well as the side of the buildings display paths drawn towards the horizon. For the perspective projection, the closer brick path seems to have the lines be more parallel than the side of the building due to the size of the building compared to the path as well as the depth of each aspect of the image. The building in the orthographic projection image possesses parallel lines because of the angle in reference to the building and the lack of a horizon and depth to the image.

Part 3: Anisotropic Diffusion Implementation

 

The third part of the was to use anisotropic diffusion to filter some images. Anisotropic Diffusion originated from Perona and Maliks journal article Scale-space and edge detection using anisotropic diffusion. The 19th figure displays the anisotropic diffusion filter apply to an image of a woman. Anisotropic diffusion possesses two diffusion functions, one being an exponential function and the other is a rational function.

Figure 20 presents the anisotropic diffusion filter being applied to an image of a CT scan using the two diffusion functions. The difference between the functions is displayed in figure 21 with the image of a CT san.

Anisotropic diffusion possesses two diffusion functions, one being an exponential function and the other is a rational function.

 

The different functions displayed in the 20th figure demonstrate that the second function seems to have more static and transitioning pixels where the first function is smoother with better visuals of the features within the image. Figures 22 and 23 contain images using anisotropic diffusion filters with a couple of parameter changes. The first parameter that was changed was the number of iterations used by the function and the second parameter was the K value. The K value is a fixed number determined by hand or noise reduction.

Figure 22: Anisotropic Diffusion with Different Number of Iterations

For both the K value and number of iterations, the trend was that the more iterations or a higher K value used, the image would have increased static and lost a lot of features with the images.
